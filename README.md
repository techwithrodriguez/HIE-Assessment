# Interoperability and HIE Assessment

Static React assessment app for student completion and Netlify hosting.

## Local Setup

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Netlify Settings

- Build command: `npm run build`
- Publish directory: `dist`

Students complete the assessment in the browser, submit it locally, review full feedback, and use **Print / Save PDF** on the completion report.
